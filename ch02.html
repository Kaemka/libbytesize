GI Overrides

The libbytesize library supports GObject introspection and thus can be used from a variety of languages. To make such use even easier and to make the code using libbytesize from different languages than C feel more natural the library also provides so-called overrides â€” pieces of code that usually wrap the original libbytesize objects and functions with language-native objects that provide language-specific features like default values for parameters etc.

The overrides are not documented here (yet), but it should be easy to directly check the sources for anybody who understands their language of choice. The overrides are located in the src/LANG_NAME directory in the source tree. The only language that libbytesize provides overrides for right now is Python (under the src/python directory).
